To run this locally, go to:

...\instagramproject\Scripts> 

in your terminal/command prompt. Then to activate the environment; write this:

...\instagramproject\Scripts>activate

There should now be a (insta_virtualenv) prior to the directory.

Then, go to:

...\instagramproject\InstagramProject\project>

and write this:

...\instagramproject\InstagramProject\project>python manage.py runserver

The server should now run on localhost:8000 and/or 127.0.0.1:8000
